Objetivos
especificos

Crear un ecosistema transmedia en el que se incluya tanto
la creación de una página web como el manejo de redes sociales,
con la finalidad de informar y poder buscar la ayuda necesaria para
quienes estén sufriendo algún tipo de violencia doméstica en Bogotá.

Objetivos
especificos


1.Generar información para que las personas que la sufren esta
violencia sea consciente.

2. Crear diferentes ayudas tras media para que estas
Víctimas sepan a dónde y cómo acudir.

3. Identificar las secuelas que quedan en una víctima para así
poder brindar ayuda por medio de voluntarios con diferentes
profesiones o entes legales.

SOBRE NOSOTROS

Núcleo Consciente es una organización que nace con el fin de ser un medio de cominucacion que busca aportar información para que las víctimas de maltrato intrafamiliar puedan acceder a ella de forma sencilla y rápida.
